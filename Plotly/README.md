# Plotly Python Library

Библиотека для создания интерактивных графиков и визуализации данных профессионального уровня. Этот репозиторий содержит подробное руководство по работе с основными модулями библиотеки.

### Установка

Для начала работы установите библиотеку через pip:

```bash
pip install plotly
```

### Содержание
Для подробного изучения конкретных инструментов перейдите в соответствующие разделы:
Plotly Express — Высокоуровневый интерфейс для быстрого создания графиков.
Graph Objects — Низкоуровневый контроль для создания сложных кастомных фигур.
Layout & Styling — Настройка внешнего вида, тем и осей.
Subplots — Работа с несколькими графиками на одной фигуре.
Interactive Tools — Кнопки, слайдеры и анимация.
Быстрый старт
### Простой пример построения графика рассеяния:

```
import plotly.express as px

df = px.data.iris()
fig = px.scatter(df, x="sepal_width", y="sepal_length", color="species")
fig.show()

```
